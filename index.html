Production-ready subscription flow

Action required: Replace placeholder price IDs (e.g., price_insightlite_99) with your real Stripe Price IDs from your Stripe Dashboard.

1. Frontend — send cart to your backend

const lineItems = cart.map(item => ({
  price: item.priceId, // e.g. 'price_1234'
  quantity: 1
}));

await fetch('/api/create-checkout-session', {
  method: 'POST',
  headers: {'Content-Type': 'application/json'},
  body: JSON.stringify({ line_items: lineItems })
});

2. Backend — create subscription session (Node.js)

const session = await stripe.checkout.sessions.create({
  mode: 'subscription',
  line_items: req.body.line_items,
  success_url: 'https://www.alleghenyairfoundation.info/success?session_id={CHECKOUT_SESSION_ID}',
  cancel_url: 'https://www.alleghenyairfoundation.info/cancel',
  allow_promotion_codes: true,
  billing_address_collection: 'auto',
  tax_id_collection: { enabled: true },
});
